[ Сценарий 1 - Регистрация пользователя ]
Пользователь вводит логин для входа в систему
Пользователь вводит пароль
Пользователь вводит адрес электронной почты
Если логин уже существует, система сообщает об этом и предлагает придумать новый
Если пароль содержит менее 6 символов, система просит ввести более длинный пароль
Если email введён некорректно, система выводит сообщение об ошибке
Если все данные корректны, система создаёт аккаунт и авторизует пользователя
Пользователь переходит к настройке целей

[Сценарий 2 - Настройка целей и уровня]
Пользователь выбирает цель тренировок: (похудение/набор формы/выносливость)
Пользователь выбирает уровень подготовки: (новичок/средний)
Пользователь выбирает доступные дни тренировок
После подтверждения система создаёт персональный план тренировок
Пользователь переходит на главный экран приложения

[Сценарий 3 - Просмотр и выполнение тренировки]
Пользователь открывает вкладку «Тренировки»
Пользователь видит план на текущий день
Пользователь нажимает «Начать тренировку»
Система показывает список упражнений с таймером и повторениями
Пользователь выполняет упражнения и отмечает выполнение
Если пользователь пропускает тренировку, система предлагает перенести её на другой день
После завершения тренировки система сохраняет результат и обновляет прогресс

[Сценарий 4 - Ввод данных беговой тренировки]
Пользователь открывает раздел «Бег»
Пользователь нажимает кнопку «Добавить пробежку»
Система открывает форму ввода данных
Пользователь вручную вводит:(дистанцию/время пробежки/дату выполнения)
При необходимости пользователь может указать комментарий (например, самочувствие или условия тренировки)
Если пользователь вводит некорректные данные (например, отрицательное число или пустое поле), система выводит сообщение об ошибке и просит исправить данные
После корректного заполнения пользователь нажимает «Сохранить»
Система сохраняет данные пробежки, обновляет статистику и отображает результат в истории тренировок

[Сценарий 5 - Трекер привычек ]
Пользователь открывает раздел «Привычки»
Пользователь нажимает «Добавить привычку»
Пользователь вводит название привычки, частоту и время напоминания
Пользователь сохраняет привычку
Привычка появляется в списке
Пользователь ежедневно отмечает выполнение привычки
Система обновляет статистику и серию выполнений
Если день пропущен, серия обнуляется
Пользователь может редактировать или удалить привычку
Система показывает общий прогресс по привычкам

[Сценарий 6 - Прогресс]
Пользователь открывает раздел «Прогресс»
Система отображает общую статистику по тренировкам, бегу и привычкам
Пользователь видит: (количество выполненных тренировок/суммарную дистанцию бега/количество выполненных привычек)
Пользователь может выбрать период (неделя/месяц)
Система обновляет статистику за выбранный период

[Сценарий 7 - Уведомления и напоминания]
Система отправляет уведомления о тренировках и привычках
Пользователь получает уведомление и может:(открыть приложение/отметить выполнение/отложить напоминание)
Если пользователь игнорирует уведомления, система повторяет напоминание позже
